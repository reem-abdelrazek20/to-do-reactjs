
React To-Do List

This is a colorful and simple To-Do List app built using React.

It lets you:

- Add tasks
- Choose a task color
- Mark tasks as done 
- Delete tasks
  
 Technologies Used

- React
- JavaScript (ES6)
- CSS
- Font Awesome (icons)
